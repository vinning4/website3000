  • What's the minimum subscription length?

    12 months minimum, then month-to-month. No setup fees, no lock-in. The initial term gives the site time to establish performance and search visibility.

  • Do I need to know anything about websites?

    Not a thing. I handle everything on the technical side and explain what I'm doing in plain English. You'll always know what's happening, and I'm reachable for any questions.

  • Why don't you use Squarespace, WordPress or Wix?

    Those platforms are built for everyone, so they're optimised for no one. They rely on themes, plugins, and databases — introducing bloat, performance drag, and security vulnerabilities. Every site I build is clean hand-written HTML and CSS: no plugins that break, no databases to hack, no glitches that frustrate customers. Which means a faster, leaner site engineered for business results. For a small business, this is the only architecture that makes sense.

  • Can I make changes after it's live?

    On the subscription plan, content changes are included — just send a message. The upfront plan includes the full source code — ideal for those who prefer to manage code independently.
